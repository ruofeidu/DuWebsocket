# DuWebsocket

## Server

The root folder contains a Node.js server. One can run through:

```bash
npm init -y
npm i ws
touch index.js
node index.js
```

Alternatively, one can use `wss://g.duruofei.com` as a deployed server.
This server is managed by PM2 and will automatically restart once crashed.
